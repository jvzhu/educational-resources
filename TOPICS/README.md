## Creating a New Topic

To add a new educational narrative topic:

1. Create a new subdirectory: `topics/your-topic-name/`
2. Add a `README.md` with topic overview
3. Add relevant resources, links, and narratives
4. Submit a pull request

**Directory Structure:**
```
topics/
├── your-topic-name/
│   ├── README.md          # Topic overview
│   ├── resources.md       # Curated links
│   └── narratives.md      # Sample narratives
```

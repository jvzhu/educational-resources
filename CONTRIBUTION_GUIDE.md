## Code of Conduct

We are committed to providing a welcoming and inclusive environment for all contributors. We expect all community members to:

- Be respectful and professional
- Welcome people of all backgrounds and experiences
- Respect different opinions and perspectives
- Focus on constructive feedback
- Celebrate diverse educational approaches

## How to Contribute

### 1. Fork the Repository
```bash
git clone https://github.com/jvzhu/educational-resources.git
cd educational-resources
```

### 2. Create a New Branch
```bash
git checkout -b feature/your-contribution-name
```

### 3. Make Your Changes
- Add, update, or improve resources
- Keep content organized and well-documented
- Follow the existing format and style

### 4. Commit Your Changes
```bash
git commit -m "Add: Description of your contribution"
```

### 5. Push to Your Fork
```bash
git push origin feature/your-contribution-name
```

### 6. Submit a Pull Request
- Provide a clear description of your contribution
- Link to any relevant issues
- Explain why this addition is valuable

## Types of Contributions

### 📚 Resource Recommendations
- Add links to educational websites, platforms, or publications
- Include descriptions and why the resource is valuable
- Specify language and target audience

### 🎙️ SoundCloud Channels
- Recommend educational podcasts or audio content
- Provide channel links and brief descriptions
- Explain how the content relates to educational narratives

### 📰 Substack Publications
- Suggest educational newsletters or columns
- Include publication links and topics covered
- Note frequency and subscription status

### 🎵 Bandcamp Artists
- Recommend educational music projects
- Describe how the music supports learning
- Provide links and artist information

### 📖 Educational Narratives
- Share your own teaching or learning stories
- Contribute narratives from other educators
- Create topic-specific narrative collections

### 🌍 New Topics
- Propose new educational narrative topics
- Create topic directories with resources
- Include overview and key concepts

### 🌐 Translations
- Help translate resources into other languages
- Ensure accuracy and cultural sensitivity
- Update resource pages with translations

### 💡 Improvements
- Fix typos and formatting issues
- Improve resource descriptions
- Enhance organization and usability
- Suggest better ways to present information

## Formatting Guidelines

### For Resource Lists
```markdown
| Resource Name | Creator/Author | Focus | Language |
|---|---|---|---|
| [Name](URL) | Creator Name | Brief description | English/Chinese |
```

### For Narrative Contributions
```markdown
## Title of Narrative
**Author:** Your Name  
**Date:** Date shared  
**Topic:** Related topic area  
**Language:** Language used

[Your narrative content here...]
```

### For New Topics
```markdown
# Topic Name
## 话题名称

Brief overview of the topic...

## Key Concepts
- Concept 1
- Concept 2

## Resources
...

## Contributing
...
```

## Pull Request Process

1. **Ensure quality**: Review your contribution for accuracy and completeness
2. **Follow format**: Maintain consistency with existing content
3. **Provide context**: Explain why this contribution is valuable
4. **Link issues**: Reference any related issues or discussions
5. **Be patient**: Maintainers will review and may suggest changes
6. **Respond to feedback**: Address any questions or requested revisions

---

## Reporting Issues

Found a broken link, inaccuracy, or have a suggestion? Please open an issue:

1. Go to [Issues](https://github.com/jvzhu/educational-resources/issues)
2. Click "New Issue"
3. Provide:
   - Clear title describing the issue
   - Detailed description
   - Steps to reproduce (if applicable)
   - Suggested solution (if applicable)
